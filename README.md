# Valor Rock — Landing Page

Single-page marketing mockup for Valor Rock, a private land partnership.
Static site — no build step. Open `index.html` directly or serve the
repository root (deploys as-is to GitHub Pages).

## Files

- `index.html` — the page
- `css/styles.css` — all styles (design tokens at the top)
- `js/main.js` — reveals, nav state, parallax, accordion, form validation
- `assets/strata-bg.jpg` — **generated placeholder.** Replace with the
  pre-blurred stone-and-copper strata artwork at the same path; no code
  changes needed.

Type system: Fraunces (display), Source Serif 4 (body), IBM Plex Mono
(technical accents — eyebrows, figures, buttons, labels), loaded from
Google Fonts.
